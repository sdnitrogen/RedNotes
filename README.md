### Features
- Offline SQLite Database
- Add, Edit, Copy, Share Notes. Add to notification for quick view.
- Search notes by keywords
- Swipe to Delete
- Limited time Trash
- Tasks with reminder
- Widgets for Notes and unchecked Tasks
- Ads and Ad-free(paid) app versions
